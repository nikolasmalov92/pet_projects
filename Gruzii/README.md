# Груз-ИИ — Telegram-бот для мониторинга грузов на ATI.SU

Бот автоматически ищет грузы на бирже ATI.SU по вашим маршрутам и фильтрам, присылает уведомления о новых и обновлённых грузах в реальном времени.

## Возможности

- **Мульти-маршрутный поиск** — несколько направлений одновременно, каждое со своими фильтрами
- **Гибкие фильтры** — вес, объём, тип загрузки, тип кузова
- **Радиус поиска** — 50 / 100 / 200 / 300 / 500 км или своё значение
- **Сохранение пресетов** — сохраняйте часто используемые маршруты с фильтрами
- **Отслеживание изменений** — бот сообщает, если у груза изменилась цена, вес или описание
- **Автообновление** — поиск каждые 3 минуты, пока вы не остановите

## Команды

| Команда | Описание |
|---|---|
| `/start` | Главное меню, регистрация, активация пробного периода |
| `/help` | Инструкция по использованию + видео-туториал |
| `/tariffs` | Тарифы и цены |
| `/subscription` | Статус вашей подписки |
| `/contacts` | Связь с администратором |

## Как начать

1. Откройте бота и нажмите `/start`
2. Нажмите **📝 Подписаться** — активируется 24-часовой пробный период
3. Нажмите **🔍 Найти грузы**
4. Настройте маршрут:
   - Выберите тип геолокации: Город / Регион / Страна
   - Введите название пункта отправления
   - Выберите радиус поиска
   - Настройте пункт назначения или выберите «Любое направление»
5. (Опционально) Настройте фильтры: вес, объём, тип загрузки, тип кузова
6. Нажмите **🔍 Начать поиск**

Бот будет присылать уведомления о новых грузах каждые 3 минуты.

## Фильтры

| Фильтр | Описание |
|---|---|
| ⚖️ Вес | Диапазон в тоннах (от / до) |
| 📦 Объём | Диапазон в м³ (от / до) |
| 🚚📦 Тип загрузки | Верхняя, боковая, задняя, полная растентовка и др. (множественный выбор) |
| 🚛 Тип кузова | Тентованный, контейнер, рефрижератор, бортовой, самосвал и др. (множественный выбор) |

## Пресеты

Часто используемые маршруты с фильтрами можно сохранить:

1. Настройте направление и фильтры
2. Нажмите **💾 Сохранить фильтр**
3. При следующем поиске выберите **📂 Мои фильтры** — загрузите сохранённый пресет

Пресеты позволяют запускать поиск по нескольким направлениям параллельно одним нажатием.

## Управление поиском

Пока поиск активен, доступны кнопки:

| Кнопка | Действие |
|---|---|
| 📊 Направления | Список активных направлений с управлением |
| ➕ Добавить | Добавить ещё направление из пресетов |
| ⛔ Остановить | Остановить все поиски |

Внутри меню направлений можно остановить, изменить или удалить каждое направление отдельно.

## Уведомления о грузах

Каждый груз приходит отдельным сообщением с информацией:
- Направление, дата, тип транспорта и загрузки
- Вес, объём, количество паллет
- Ставка (с НДС / без НДС) и ставка за км
- Габариты (ДxШxВ), расстояние
- Название компании, примечание
- Кнопка **📋 Подробнее** — ссылка на полную карточку на ATI.SU

Если у груза изменилась цена или описание — бот пришлёт обновление с выделением изменений.
