# 🐞 Отладка реальности

## 📂 Список проектов

| Эмодзи | Проект | Что делает | Суперспособность |
|--------|--------|------------|------------------|
| ⏰ | **Голосовой будильник** | В 8:00 рассказывает погоду, пробки, новости и время до работы | Работает на **Orange Pi** + колонки, будит голосом |
| ✈️ | **Билетолёт** (Avia Price Bot) | Telegram-бот для дешёвых авиабилетов | Ловит билеты, как коршун — цены ниже рынка |
| 🚛 | **Груз-ИИ** | Ищет грузоперевозки по направлениям | Умная фильтрация предложений, секундный отклик |
| 🌡️ | **home_eco_climat** | Управляет кондиционером Dahatsu | Экономит электричество и не даёт замёрзнуть |
| 🔌 | **send_serial_port** | Отправляет данные через COM-порт | Работает с любым железом через последовательный порт |
| 💬 | **whatsapp_bot** | Бот-отправитель в WhatsApp | Автоматические уведомления в мессенджер |
